=========================================
Score Batcher Documentation
=========================================
Smarter Score Batcher is a system designed for processing and batching real-time assessment results documents. 
The batcher consists of the RESTful API to receive the results, distributed service to process the results, and
a bacther service to produce batched up assessment results for further ETL.
